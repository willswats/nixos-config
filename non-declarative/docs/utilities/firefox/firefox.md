## Extension Settings

### Bitwarden

- Ask to add login: Disabled
- Ask to update existing login: Disabled

### uBlock

- Run in Private Windows: Allow

### Consent-O-Matic

- Preferences and Functionality: Enabled

### MAL-Sync

- Notifications (Anime): Disabled
