# osu

## Options

Add `non-declarative/configs/osu/game.ini` to `.local/share/osu`.

## Skins

Drag and drop all of the skins into osu (stored in my personal drive).

### Keybindings

#### Global

Back (2): `BACKSPACE`
Quick Retry (hold): `RSHIFT`

#### osu (section)

Left button (2): `,`
Right button (2): `.`
Smoke: `/`
