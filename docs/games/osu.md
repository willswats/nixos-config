# osu

## Skins

Run all skin files.

## Options

Add `game.ini`, `framework.ini` and `input.json` to `.local/share/osu`.

### Keybindings

#### Global

Back (2): `BACKSPACE`
Quick Retry (hold): `RSHIFT`

#### osu (section)

Left button (2): `,`
Right button (2): `.`
Smoke: `/`

## Beatmaps

Add `files` to `.local/share/osu`.
