# Slippi

## Slippi Launcher

Download:

```bash
wget https://github.com/project-slippi/slippi-launcher/releases/latest/download/Slippi-Launcher-2.10.2-x86_64.AppImage -P ~/AppImages
```

Sign in and provide Melee ISO.

## Slippi Online

To run:

```bash
slippi-online
```
